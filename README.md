# Mindful Haven

Mindful Haven is a fullstack web application built with modern web technologies.

## Tech Stack

### Frontend
- React 19
- Vite
- TailwindCSS
- Radix UI
- React Query

### Backend
- Express 5
- Drizzle ORM
- PostgreSQL
- Passport Authentication

## Features
- REST API integration
- Authentication system
- Modern UI components
- Type-safe database queries
- Fullstack development setup

## Installation

```bash
npm install
npm run dev
